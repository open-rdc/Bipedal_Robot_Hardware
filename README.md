# Bipedal_Robot_Hardware

RDC_LAB ２足歩行ロボット開発リポジトリ（ハードウェア）


## Circuit
- RDC_Bipedal_Logic
  - Logic回路基板。コンピュータや通信周り
  - Kicad Version: 7
- RDC_Bipedal_Power_Relay_Switch
  - Power回路のリレーのスイッチと、非常停止ボタンの回路
  - Kicad Version: 7
- RDC_Bipedal_Power
  - Power回路図。基板はなく回路図のみ
  - Kicad Version: 9
